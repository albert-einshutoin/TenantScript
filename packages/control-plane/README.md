# @tenantscript/control-plane

TenantScript Control Plane domain services, Admin HTTP contracts, storage adapters, RBAC, audit,
usage, execution retention, and Cloudflare Worker composition primitives.

See the [threat model](https://github.com/albert-einshutoin/TenantScript/blob/main/docs/security/threat-model.md)
and [Control Plane error catalog](https://github.com/albert-einshutoin/TenantScript/blob/main/docs/reference/control-plane-errors.md)
before exposing a new route or persistence adapter.
