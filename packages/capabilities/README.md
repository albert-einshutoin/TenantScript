# @tenantscript/capabilities

Tenant-scoped capability contracts and provider adapters for TenantScript plugins. The broker keeps
provider credentials outside plugin code and enforces the granted operation and resource scope.

See the [provider adapter reference](https://github.com/albert-einshutoin/TenantScript/blob/main/docs/reference/provider-adapters.md)
and the repository [security policy](https://github.com/albert-einshutoin/TenantScript/blob/main/SECURITY.md)
before implementing a provider.
