# @tenantscript/cli

The TenantScript `ext` command-line interface and its typed integration helpers. It supports plugin
scaffolding, build/dev/replay, schema and manifest checks, dry-run deploy, self-host planning,
diagnostics, and authenticated Admin mutations.

See the [CLI reference](https://github.com/albert-einshutoin/TenantScript/blob/main/docs/reference/cli.md)
for implemented commands and operational boundaries. Live Cloudflare setup remains explicitly
separated from accountless planning.
