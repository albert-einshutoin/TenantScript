# @tenantscript/plugin-sdk

Plugin-author SDK helpers for defining typed TenantScript plugins against the public manifest and
hook contracts.

Follow the [SDK integration quickstart](https://github.com/albert-einshutoin/TenantScript/blob/main/docs/quickstarts/sdk-integration.md)
to create, test, bundle, validate, and dry-run a plugin without production credentials.
