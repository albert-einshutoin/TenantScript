# @tenantscript/proxy

TenantScript zero-integration proxy mode for validating inbound webhooks, running a scoped hook, and
forwarding only to an explicitly allowed public origin.

Use the [proxy mode quickstart](https://github.com/albert-einshutoin/TenantScript/blob/main/docs/quickstarts/zero-integration-proxy-mode.md)
and retain the SSRF, tenant-scope, signature, and response-limit protections when integrating it.
