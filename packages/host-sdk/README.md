# @tenantscript/host-sdk

Host-side SDK contracts for registering plugin versions, installations, approvals, and typed hook
execution in an existing SaaS application.

Start with the [SDK integration quickstart](https://github.com/albert-einshutoin/TenantScript/blob/main/docs/quickstarts/sdk-integration.md)
and preserve the authenticated app and tenant scope when adapting storage or transport.
